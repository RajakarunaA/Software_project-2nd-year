# Project Management Tool - 2nd Year Software Project

This is a **Project Management Tool** developed as part of a 2nd-year Software Project. It is a full-stack web application that allows users to manage projects, tasks, and team members efficiently. The application includes a **React frontend** and a **Node.js + Express + MySQL backend**. Here is my contribution **Admin Dashboard.**

## Overview

The **Project Management Tool-Admin Pannel** enables users to:

- View, add, update, and delete projects and tasks
- Manage users and team members
- Track project progress efficiently
- Demonstrate full-stack development concepts

This project was created to provide hands-on experience in **software development, database integration, and frontend-backend communication**.

---

## Features

### Project & Task Management
- Create, update, delete, and view projects
- Assign tasks to team members
- Track project details such as deadlines, descriptions, and status

### User Management
- Register and manage users
- Update user information
- Delete users as needed

## Technology Stack

**Frontend:**
- React
- React Router v6
- Axios (for API calls)
- CSS

**Backend:**
- Node.js
- Express.js
- MySQL
- CORS
